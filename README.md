## 📚 About

This repository contains my solutions, exercises, and notes for the **Full Stack Open 2026** course offered by the **University of Helsinki**.

The course covers modern web development with:

- React
- TypeScript
- Node.js
- Express
- GraphQL
- MongoDB
- Testing
- CI/CD
- Docker
- Kubernetes
